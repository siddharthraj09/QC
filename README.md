
#  QC-DIGITAL Monorepo

Assignment by siddharth 


## Run Locally

Clone the project

```bash
  git clone https://github.com/siddharthraj09/QC
```

Go to the project directory

```bash
  cd qc_digital
```

Install dependencies

```bash
  npm install
  npm install turbo --global
```

Start the server

```bash
 turbo dev
```
